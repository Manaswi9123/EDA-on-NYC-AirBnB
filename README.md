# EDA-on-NYC-AirBnB
# 🗽 NYC Airbnb Market Analysis - Exploratory Data Analysis (EDA)

This project provides a comprehensive data-driven look into the New York City Airbnb market. By analyzing price trends, location popularity, and host behavior, the project identifies which areas of NYC offer the most value and how different room types impact rental costs.

---

## 🛠️ The Tech Stack
* **Python**: Core programming.
* **Pandas**: Data cleaning and neighborhood grouping.
* **NumPy**: Statistical calculations.
* **Matplotlib & Seaborn**: For heatmaps, boxplots, and geospatial visualizations.

---

## 📊 Project Highlights

### 1. Data Cleaning & Integrity
* Handled missing values in critical columns like `reviews_per_month` and `name`.
* Filtered out price outliers to ensure the visualizations reflect the actual market reality.

### 2. Neighborhood & Borough Analysis
* **Borough Breakdown:** Compared listing densities and average prices across Manhattan, Brooklyn, Queens, the Bronx, and Staten Island.
* **Top Neighborhoods:** Identified the most expensive and the most affordable neighborhoods for travelers.

### 3. Room Type & Availability
* Analyzed the distribution between **Entire homes/apartments**, **Private rooms**, and **Shared rooms**.
* Investigated host "busyness" by analyzing the number of reviews and minimum stay requirements.

### 4. Pricing Insights
* Used **Box Plots** to visualize price variances across different regions.
* Created **Scatter Plots** using latitude and longitude to map out price density across NYC's geography.
